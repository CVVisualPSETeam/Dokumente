hier können  klassen eingetragen werden (+ was sie machen)

- indexer
	- eine klasse die ein indexarray verwaltet, welches einen vektor indiziert 
	- dient uns dazu unsere daten nicht kopieren zu müssen
	- kann funktion anbieten um an den orginalen index zu kommen
	- kann z.b. sortiert werden (die orginale datenstruktur wird nicht verändert!)
	
- historgramSelecter 
	- ein histogramm in derm ein bereich ausgewählt werden kann
	
- Match
	- ein ersatz für die matches aus opencv
	- einthält nicht nur den index auf die entsprechenden Punkte sondern auch diese/ eine referenz auf diese
